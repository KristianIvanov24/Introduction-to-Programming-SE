# Бройни системи. Побитови операции.

# В час

1. Напишете функция, която приема масив (разглеждаме го като множество) и отпечатва всички негови подмножества.

<details> <summary>Примерен вход и изход</summary>

Вход:
```c++
[1, 2, 3]
```
Изход:
```c++
[], [1], [2], [1,2], [3], [1,3], [2,3], [1,2,3]
```
</details>

2. Напишете функция, която приема масив, в който всички числа се срещат четен брой пъти освен едно от тях. Функцията да връща кое е това число.

<details> <summary>Примерен вход и изход</summary>

Вход:
```c++
[1, 2, 10, 3, 2, 3, 1]
```
Изход:
```c++
10
```
</details>

# Задачи

1. Напишете функция, която приема число `k`, (`0 < k <= 16`) и масив от символи репрезентиращи числото в `k`-ична бройна система и приема число `m`, (`0 < m <= 16`). Функцията извежда на стандартния изход числото в `m`-ична бройна с-ма.

<details> <summary>Примерен вход и изход</summary>

Вход:
```c++
k = 12
m = 10
['1', '1', 'B']
```
Изход:
```c++
167
```

Вход:
```c++
k = 4
m = 16
['2', '0', '2', '6']
```
Изход:
```c++
8E
```

</details>

2. Да се напише функция, която приема 3 цели числа `n`, `pos` - позиция и бит `b` - 0 или 1 и сменя бита на `n` на позиция `pos` с бита `b`.

<details> <summary>Примерен вход и изход</summary>

Вход:
```c++
120 2 1
```
Изход:
```c++
124
```

Вход:
```c++
120 4 0
```
Изход:
```c++
104
```
</details>

3. Напишете функция, която извежда на екрана двоичната репрезентация на 8 битово неотрицателно число, записано в десетична бройна система.

<details> <summary>Примерен вход и изход</summary>

Вход:
```c++
2
```
Изход:
```c++
00000010
```

Вход:
```c++
137
```
Изход:
```c++
10001001
```
</details>

4. Напишете функция, която приема 2 цели числа `a` и `b` и проверява дали битовете (вкл. и водещите нули, ако има такова) на `a` са пермутация на битовете на `b`.

</details>

```c++
Monday, Thursday, Saturday, Sunday
```

6. Да се напише функция, която приема число от тип `unsigned int` (32 бита) и `k`, (`0 < k <= 16`). Функцията да връща дали числото е палиндром в дадената бройна система.

<details> <summary>Примерен вход и изход</summary>

Вход:
```c++
7 2
```
Изход:
```c++
true
//7 = 111 in binary
```
</details>


7. Напишете функция, която приема 3 цели числа `num`, `pos` и `k` и връща числото, което се получава, ако от num се вземат `k` бита, започвайки от позиция pos.

<details> <summary>Примерен вход и изход</summary>

Вход:
```c++
num = 16 
pos = 4 
k = 3
```
Изход:
```c++
4 // 16 = 10000 => |100|00 => 100 = 4
```

Вход:
```c++
3 // 0011
8 // 1000
```
Изход:
```c++
0
```
</details>

8. Напишете функция, която приема два масива от символи, големината им (която е еднаква), число `k` и празен масив `res` заедно с големината му. Масивите от символи са числа в `k`-ична бройна система. Функцията трябва да прибавя второто към първото и да попълни резултата в `res`.

<details> <summary>Примерен вход и изход</summary>

Вход:
```c++
[1, 7, 5] 
[5, 3, 7]
3 8
[]
```
Изход:
```c++
[7, 3, 4]
```
</details>
